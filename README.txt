This readme provides the basic steps to compile and run each command in the command line

***********************************************************************
mymv

To compile:
g++ -o mymv FileManager.cpp mymv.cpp
OR
make mymv

To run:
./mymv <parameters>

***********************************************************************

***********************************************************************
To compile all of the files, run this block of commands in the terminal:

g++ -o mymv FileManager.cpp mymv.cpp
g++ -o mycp FileManager.cpp mycp.cpp
g++ -o myls FileManager.cpp myls.cpp
g++ -o mycat FileManager.cpp mycat.cpp
g++ -o myrm FileManager.cpp myrm.cpp
g++ -o mydiff FileManager.cpp mydiff.cpp
g++ -o mystat FileManager.cpp mystat.cpp

OR
make

***********************************************************************
To delete all of the executable files, run:

make clean
